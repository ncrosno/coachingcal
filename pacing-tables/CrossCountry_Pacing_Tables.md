# Cross Country Training Pace Tables
*Based on current 1600m PR - Updated with Modern Exercise Science 2025*

**How to Use These Tables:**
1. Find your current 1600m PR in the left column
2. Use the corresponding pace for each type of workout
3. Zone 2 = Easy/recovery runs (conversational)
4. LT1 = Moderate tempo (aerobic threshold)  
5. LT2 = Hard tempo (lactate threshold)
6. 5K Race Pace = Race simulations and intervals
7. VO2max = Short, hard intervals (3-8 minutes)

**Important Notes:**
- All paces are per mile
- Adjust for heat/humidity conditions
- Listen to your body and use RPE as backup
- HRV monitoring can override these prescriptions
